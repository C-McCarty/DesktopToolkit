# Legacy — original Python Monitor Arrangement

These are the original Python 3 / tkinter sources that seeded this project. They have
been **superseded** by the C#/WPF **Monitor Arrangement** module
(`src/Modules/MonitorArrangement`), which has feature parity (drag/snap/grid, sidebar,
undo/redo, apply, profiles, identify) and runs inside the Desktop Toolkit suite.

They are kept for reference only. To run the old version:

```
cd legacy
python main.py
```
